# chaiscript-cli-hello-world

## Description
A POC for chaiscript programming language.

## Tech stack
- cmake
- g++
- git
- make

## Docker stack
- ubuntu:22.04
- rzuckerm/chaiscript

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
